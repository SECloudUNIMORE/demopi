# GPS Spoofing attack demo
Demonstrate a GPS spoofing attack.

## Installation
0. setup the base system as per `../../README.md` instructions
1. run the Ansible Playbook:
   ```sh
   ansible-playbook -i ../../inventory --user <USER> gps-spoof.yml
   ```
